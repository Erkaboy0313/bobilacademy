Backend dasturlash kursi

Backend sayt va serverning ichki qismidir (ing. Orqaga - orqaga). Umuman olganda, bu sayt va serverning to'g'ri ishlashiga imkon beruvchi dasturiy ta'minot va apparat majmuasidir.

Kurs davomida quyidagi texnalogiyalar o'tiladi

  • Python

    Python - bu turli sohalar uchun yuqori darajadagi umumiy maqsadli dasturlash tili.

  • OOP

    OOP - dasturlashda o’zaro bo’gliq bo’lgan o’zgaruvchilar va funksiyalar bitta konteynerga jamlanadi va bunday konteynerlar obyekt deb ataladi

  • PostgreSQL

    PostgreSQL - bu relyatsion ma'lumotlar omborini boshqarish tizimi. Ya'ni, ma'lumotlarni boshqarish tizimi bo'lib, u aloqalar(relation) ko'rinishida taqdim etiladi.

  • Telegrab Bot

    Bot - bu haqiqiy odamning mavjudligini taqlid qiluvchi dastur (virtual suhbatdosh). Telegram bot mos ravishda yozishmalarni almashtirishga qodir bo'lgan ilova.

  • GitHub

    GitHub - bu Git repozitoriylarini boshqarishga imkon beradigan bulutga asoslangan hosting xizmati.

  • Django

    Django - python dasturlash tilida veb ilovalar ishlab chiqish uchun ajoyib freymvork.

  • RestAPI

    API - serverning asosiy qismi bo'lib, u so'rovlar qabul qiladi va ularga javoblar qaytaradi.

  • Real loyihalar bilan ishlash
