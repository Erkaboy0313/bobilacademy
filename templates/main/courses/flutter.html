Flutter dasturlash kursi

Flutter - Google tomonidan ishlab chiqilgan framework hisoblanadi. Bu kurs orqali siz bir vaqtning o'zida yangi til Dart ni o`rganib Android va iOS mobil ilovalarni yarata olasiz.

Kurs davomida quyidagi texnalogiyalar o'tiladi

  • Dart

    Dart - Google kompaniyasi tomonidan, umumiy qo'llanish uchun yaratilingan dasturlash tilidir.

  • Firebase

    Firebase - bu real vaqtda ma'lumotlar bazasida fayllaringizni saqlaydigan bepul hosting .

  • Sqflite

    Sqflite local datrabase bolib malumotlarni jadval korinishida saqlaydi

  • Hive

    Hive bu local database bolib malumotlaringizni local holatda saqlah uchun kerak boladigan package

  • Provider

    Provider : inheretent widget qilingan holati bolib bir sahifada ozgarishlarni qayta tetiklash uchun kerak boladigan package

  • Bloc

    Bloc: State Managment mahsuloti bolib sahifada boladigan ozgarishlarni boshqarish uchun kerak boladigan package

  • Getx

    Getx bu State Managment mahsuloti bolib sahifadagi holatlarni qayta tetiklash uchun va birqancha qulaylikarga ega Flutter package.

  • UI Clone Instagram

    Quyidagi texnalogilaridan foydalanib Android, IOS tizimlariga Instagnam ilovasini yaratish.
